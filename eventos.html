Eventos del Mouse en JavaScript

Los eventos son acciones que suceden en el navegador. JavaScript puede "escuchar" estos eventos para crear interactividad. Aquí exploramos los más comunes aplicados a botones.

1. onclick

El evento más fundamental. Se dispara cuando el usuario presiona y suelta el botón izquierdo del mouse sobre el elemento.

Uso común: Enviar formularios, abrir enlaces, confirmar acciones.

2. ondblclick

Se activa solo cuando el usuario hace dos clics rápidos y sucesivos sobre el mismo elemento.

Uso común: Abrir archivos, editar texto, hacer zoom.
Esperando...

3. onmouseover

Se dispara en el momento exacto en que el puntero del mouse entra en el área del botón.

Uso común: Mostrar tooltips (ayudas), efectos hover, menús desplegables.
Esperando...

4. onmouseout

Lo opuesto al anterior. Se dispara cuando el puntero del mouse sale del área del botón.

Uso común: Ocultar menús, revertir colores, cerrar ayudas.
Esperando...

5. onmousedown

Se activa inmediatamente cuando se presiona el botón del mouse, antes de soltarlo.

Uso común: Juegos (disparar), arrastrar y soltar (drag & drop).
Esperando...

6. onmouseup

Se activa cuando el usuario suelta el botón del mouse después de haberlo presionado.

Uso común: Finalizar una acción de arrastre, detener un cronómetro.
