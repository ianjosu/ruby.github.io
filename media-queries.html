Media Queries

Las Media Queries son el corazón del diseño web responsivo (Responsive Design). Permiten que una página web cambie su apariencia y estructura dinámicamente según el tamaño de la pantalla, la orientación o la resolución del dispositivo.

1. Conceptos Fundamentales

@media

La Regla @media

Es la instrucción CSS que le dice al navegador: "Si se cumple esta condición (por ejemplo, la pantalla mide menos de 600px), aplica estos estilos nuevos".

min-width

Min-Width (Desde...)

Define estilos que se aplicarán a partir de cierto tamaño. Es la base del enfoque "Mobile First" (Diseñar primero para móvil y luego expandir).

max-width

Max-Width (Hasta...)

Define estilos que se aplicarán hasta cierto tamaño máximo. Útil para hacer ajustes específicos en pantallas pequeñas (Desktop First).

breakpoints

Breakpoints

Son los "puntos de quiebre" o medidas estándar donde el diseño cambia. Comúnmente: 576px (móvil), 768px (tablet), 1024px (desktop).

2. Estructura de Código

Así es como se ve una Media Query típica en CSS:

/* Estilos Generales (Móvil por defecto) */
body {
    background-color: lightblue;
    font-size: 14px;
}

/* Para Tablets (mínimo 768px) */
@media (min-width: 768px) {
    body {
        background-color: lightgreen;
        font-size: 16px;
    }
}

/* Para Escritorio (mínimo 1024px) */
@media (min-width: 1024px) {
    body {
        background-color: salmon;
        font-size: 18px;
    }
}
