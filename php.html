Guía Definitiva de PHP

¿Qué es PHP?

PHP es un lenguaje de programación de código abierto diseñado para el desarrollo web del lado del servidor. Se ejecuta en el servidor y genera contenido HTML dinámico.

¿Por qué usar PHP?

  • Lenguaje ampliamente usado en la web
  • Compatible con múltiples servidores
  • Integración directa con HTML
  • Soporte nativo para MySQL

Delimitadores de PHP

Tipo Sintaxis Uso
Estándar <?php ... ?> Forma recomendada
Echo corto <?= $variable ?> Mostrar valores 
<!DOCTYPE html>
<html>
<body>
<?php
echo "Hola desde PHP";
?>
</body>
</html>

Variables

<?php
$nombre = "Juan";
$edad = 20;
$activo = true;
echo $nombre;
?>

Estructuras de Control

  • if / else / elseif
  • switch
  • for
  • while
  • foreach

Arrays Asociativos

<?php
$frutas = ["manzana" => 5, "uva" => 10];
foreach ($frutas as $nombre => $cantidad) {
    echo $nombre . ": " . $cantidad;
}
?>

Funciones

<?php
function saludar($nombre) {
    return "Hola " . $nombre;
}
echo saludar("Carlos");
?>

Inclusión de Archivos

  • include incluye archivos no críticos
  • require detiene la ejecución si falla

Bases de Datos

PHP se conecta a MySQL mediante MySQLi o PDO usando sentencias preparadas para mayor seguridad.
