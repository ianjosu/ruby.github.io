HTML: Tipos de Listas

Organización, Jerarquía y Estructura

Las listas son esenciales para agrupar contenido relacionado. HTML nos proporciona tres tipos principales para estructurar la información: Desordenadas, Ordenadas y de Descripción.

1. Listas Desordenadas (<ul>)

Se utilizan cuando el orden de los elementos no importa. Por defecto, usan viñetas (puntos).

<ul>
    <li>HTML 5</li>
    <li>CSS 3</li>
    <li>JavaScript</li>
</ul>
  • HTML 5
  • CSS 3
  • JavaScript

2. Listas Ordenadas (<ol>)

Se usan cuando la secuencia es importante (pasos, rankings). Usan números o letras.

Podemos cambiar el tipo con el atributo type ("1", "A", "a", "I", "i").

<ol type="I"> <!-- Números Romanos -->
    <li>Planificar</li>
    <li>Diseñar</li>
    <li>Programar</li>
</ol>
  1. Planificar
  2. Diseñar
  3. Programar

3. Listas de Descripción (<dl>)

Perfectas para glosarios o diccionarios. Consisten en un Término (dt) y una Descripción (dd).

<dl>
    <dt>Frontend</dt>
    <dd>Lo que el usuario ve.</dd>
    
    <dt>Backend</dt>
    <dd>La lógica del servidor.</dd>
</dl>
Frontend
La parte visual e interactiva que ve el usuario.
Backend
La lógica, bases de datos y servidor que no se ve.

4. Listas Anidadas

Podemos poner una lista dentro de otra para crear subniveles o menús.

<ul>
    <li>Frutas
        <ul>
            <li>Manzana</li>
        </ul>
    </li>
</ul>
  • Frutas
    • Manzana
    • Pera
  • Verduras
    • Zanahoria
