Compendio de Lenguajes

Guía rápida de tecnologías esenciales para el desarrollo moderno.

1. Desarrollo Web Frontend

Tecnología Función Principal Sintaxis Rápida
HTML .html El esqueleto de la web. Define la estructura y el contenido semántico. 
<h1>Hola Mundo</h1>
CSS .css El maquillaje de la web. Controla diseño, colores, y animaciones. 
body { color: #00ffff; }
JS .js El cerebro de la web. Añade interactividad y lógica dinámica. 
alert("Bienvenido");

2. Backend y Datos

Lenguaje Descripción Sintaxis Rápida
PHP .php Líder en servidores web. Ideal para crear sitios dinámicos y CMS. 
<?php echo "Hola"; ?>
Python .py Versátil y poderoso. Usado en IA, Ciencia de Datos y Backend. 
print("Hola Python")
Java .java Robusto y orientado a objetos. Estándar en grandes empresas. 
System.out.print("Hi");
SQL .sql Lenguaje de consulta para interactuar con Bases de Datos. 
SELECT * FROM usuarios;

3. Sistemas y Móvil

Lenguaje Enfoque Sintaxis Rápida
C++ .cpp Alto rendimiento. Motores de videojuegos y software complejo. 
cout << "Hola Mundo";
Swift .swift Nativo de Apple. Para crear apps de iPhone, iPad y Mac. 
print("Hola iOS")
Kotlin .kt El estándar moderno para desarrollo de aplicaciones Android. 
println("Hola Android")
