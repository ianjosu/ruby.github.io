CSS: El Arte del Estilo

Cascading Style Sheets - La piel de la web

CSS es el lenguaje utilizado para describir la presentación de un documento HTML. Mientras que HTML es la estructura (los huesos), CSS es el estilo (la piel, la ropa y el maquillaje).

1. Sintaxis Básica

Una regla CSS se compone de un selector y un bloque de declaración. 

selector {
    propiedad: valor;
    color: #ff007f;
}

Tipos de Selectores

Selector Ejemplo Descripción
Etiqueta p { } Afecta a todos los párrafos.
Clase .boton { } Afecta a elementos con class="boton". (Reutilizable)
ID #header { } Afecta a un elemento único con id="header".
Universal * { } Afecta a todos los elementos de la página.

2. El Modelo de Caja (Box Model)

Todo en CSS es una caja rectangular. Entender esto es la clave del diseño. Una caja tiene 4 capas:

  • Content: El texto o imagen real.
  • Padding: Espacio interior (entre el contenido y el borde).
  • Border: El borde alrededor del padding.
  • Margin: Espacio exterior (separa esta caja de otras).

3. Sistemas de Diseño (Layouts)

Antiguamente se usaban "floats", pero hoy dominan dos gigantes:

A. Flexbox (Unidimensional)

Perfecto para alinear elementos en una fila o una columna. Esencial para menús y centrado vertical.

.contenedor {
    display: flex;
    justify-content: space-between; /* Separa los items */
    align-items: center; /* Centra verticalmente */
}

B. CSS Grid (Bidimensional)

El sistema más potente. Permite crear filas y columnas simultáneamente.

.contenedor {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
    gap: 10px; /* Espacio entre celdas */
}

4. Diseño Responsivo

Hacer que la web se vea bien en celulares y computadoras. Se usa la regla @media. 

/* Estilos base (Móvil primero) */
body { font-size: 14px; }

/* Para pantallas mayores a 768px (Tablets/PC) */
@media (min-width: 768px) {
    body { font-size: 18px; }
    .menu { display: flex; }
}

5. Unidades y Colores

Unidades de Medida

  • px: Píxeles (Fijo).
  • %: Porcentaje del padre.
  • rem: Relativo al tamaño de fuente raíz (Recomendado para accesibilidad).
  • vh / vw: Porcentaje del tamaño de la pantalla (Viewport).

Colores

  • Hexadecimal: #ff007f (Lo que usamos aquí).
  • RGB: rgb(255, 0, 127).
  • RGBA: rgba(255, 0, 127, 0.5) (Con transparencia).
