JavaScript: El Cerebro

Interactividad, Lógica y Dinamismo Web

Si HTML es la estructura y CSS es la apariencia, JavaScript (JS) es el cerebro y los músculos. Es un lenguaje de programación que permite crear contenido dinámico, controlar multimedia, animar imágenes y modificar tu web en tiempo real.

1. Variables

Son contenedores para guardar datos. En JS moderno usamos principalmente let y const.

let nombre = "Eon Flux";  // Se puede cambiar después
const PI = 3.1416;        // No se puede cambiar (Constante)
var antiguo = "Hola";     // Manera antigua (Evitar usar)

2. Tipos de Datos

  • String: Texto. "Hola Mundo"
  • Number: Números. 25 o 10.5
  • Boolean: Verdadero/Falso. true o false
  • Array: Listas. ["HTML", "CSS", "JS"]
  • Object: Colecciones de datos. { nombre: "Alex", edad: 20 }

3. Funciones

Bloques de código diseñados para realizar una tarea particular.

// Función Tradicional
function saludar(nombre) {
    return "Hola " + nombre;
}

// Función Flecha (Moderna)
const sumar = (a, b) => a + b;

4. El DOM (Document Object Model)

Es la representación de tu HTML que JavaScript puede ver y modificar. Es como JS "agarra" elementos de la pantalla. 

// Seleccionar un elemento por su ID
let titulo = document.getElementById("miTitulo");

// Cambiar el texto
titulo.innerText = "Nuevo Título";

// Cambiar el estilo
titulo.style.color = "#ff007f";

5. Eventos

Cosas que suceden en el sistema (clics, teclas, scroll). JS "escucha" estos eventos para reaccionar.

boton.addEventListener('click', () => {
    alert("¡Hiciste clic!");
});
